# 🚀 AI-Powered Employee Attrition Intelligence System

## Overview

An end-to-end Machine Learning and MLOps solution that predicts employee attrition risk and helps HR teams make proactive retention decisions. The system integrates automated data pipelines, MLflow model management, FastAPI APIs, and a Streamlit dashboard for real-time predictions.

## Key Features

* Employee Attrition Prediction
* Individual Employee Risk Analysis
* Batch CSV Prediction
* Data Validation & Feature Engineering Pipelines
* MLflow Experiment Tracking & Model Registry
* FastAPI REST API Deployment
* Interactive Streamlit Dashboard
* Production Model Versioning

## Tech Stack

**Python | Scikit-Learn | Pandas | NumPy | MLflow | FastAPI | Streamlit | Git | GitHub**

## Architecture

```text
Dataset
   ↓
Data Validation
   ↓
Feature Engineering
   ↓
Model Training
   ↓
MLflow Registry
   ↓
FastAPI API
   ↓
Streamlit Dashboard
```

## API Endpoints

* `GET /` → Health Check
* `POST /predict` → Individual Prediction
* `POST /batch_predict` → Batch Prediction

## Business Impact

* Predicts employee attrition risk
* Supports data-driven HR decisions
* Improves employee retention strategies
* Reduces recruitment and training costs
